### REQUIREMENT

## Hardware

1. Arduino Uno board
2. LED light
3. Push button
4. Resistor (2.2 Kohms)

## Software

1. 'Dino' gem
2. Ruby -v 2.2.2
3. Rails -v 4.2.6

## Installation

1. gem install dino
2. dino generate-sketch serial (this will create a folder named 'du')
3. open `.../du/du.ino` file in Arduino software
4. select appropriate board and port from Arduino software
5. upload file to Arduino board
6. run Ruby program from terminal

see [Getting Started with Arduino and Dino](http://tutorials.jumpstartlab.com/projects/arduino/introducing_arduino.html)
